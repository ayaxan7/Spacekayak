# Spacekayak (Android)

A Jetpack Compose Android app. This repo includes Compose UI, Navigation, and accompanist-pager for onboarding.

## Quick start

Prerequisites
- Android Studio (Ladybug or newer)
- Android SDK (API 28+), Gradle wrapper is included
- JDK 11 (project is configured for Java 11)

Run with Android Studio
1. Open this folder in Android Studio
2. Let it sync/resolve Gradle
3. Choose a device/emulator and click Run â–¶
